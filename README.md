# Clare Facio

Website for Clare Facio, a Costa Rican law firm. Modern, multilingual (ES/EN/FR), single-page landing site.

**Stack:** Next.js 16 · TypeScript · Tailwind CSS v4 · Framer Motion · next-intl · Vercel

## Dev

```bash
pnpm dev      # start local server at http://localhost:3000
pnpm build    # production build
pnpm lint     # run linter
```

On first visit, the site auto-detects browser language and redirects to `/es`, `/en`, or `/fr`.
